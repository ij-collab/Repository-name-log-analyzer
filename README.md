# Python Log Analyzer


## Overview
This project is a simple log analyzer written in Python.

It reads a system log file and counts important log entries.

## Features

- Read log files
- Count log levels
- Save analysis results

## Files

- log_analyzer.py
- system.log
- analysis_result.txt

## How to Run

```bash
python log_analyzer.py
```
